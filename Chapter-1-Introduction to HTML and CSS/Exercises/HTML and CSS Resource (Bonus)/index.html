HyperText Markup Language (HTML)
<!--...--> This defines a comment
<!DOCTYPE> The header tag defines the document type
<body> The body tag defines the document's body
<header> The header tag defines a header for a document or section
<footer> The footer tag defines a footer for a document or section
<head> The head tag contains metadata/information for the document
<style> the style tag defines style information for a document
<p> The p tag defines a paragraph
<h1> to <h1> These tags define HTML headings
<ul> The ul tag defines an unordered list
<ol> The ol tag defines an ordered list
<li> The li tag defines a list item
<pre> the pre tag defines preformatted text
<img> the img tag defines style an image
<video> The video tag defines embedded video content
<audio> The audio tag defines embedded sound content
<a> the a tag defines a hyperlink
<br> The br tag defines a single line break
<button> The button tag defines a clickable button
<nav> the nav tag defines navigation links
<form> The form tag defines an HTML form for user input
<input> The input tag defines an input control
<label> the label tag defines a label for an <input> element
<div> The div tag defines a section in a document
<table> The table tag defines a table
Cascading Stylesheet (CSS)
/*...*/ This defines a comment
color The color property sets the color of text
background-color The background-color property specifies the background color of an element
background-image The background-image property specifies one or more background images for an element
background-attachment The background-attachment property sets whether a background image scrolls with the rest of the page, or is fixed
border The border property is a shorthand property for border-width, border-style and border-color
font-size The font-size property specifies the font size of text
font-family The font-family property specifies the font family for text
margin The margin property sets all the margin properties in one declaration
padding The padding property is a shorthand property for all the padding-* properties
position The position property specifies the type of positioning method used for an element (static, relative, absolute or fixed)
display The display property specifies how a certain HTML element should be displayed
text-decoration The test-decoration property specifies the decoration added to text
border-radius The border-radius property is a shorthand property for the four border-*-radius properties
text-align The display property specifies how a certain HTML element should be displayed
float The test-decoration property specifies the decoration added to text
border-spacing The border-spacing property sets the distance between the borders of adjacent cells
line-height The line-height property sets the line height
flex The flex property is a shorthand property for the flex-grow, flex-shrink, and the flex-basis properties
@font-face The border-radius property specifies a custom font to use to display text
